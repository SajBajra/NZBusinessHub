GeoDirectory Event Tickets.

== Description ==

The GeoDirectory Events Tickets Marketplace extension allows you to easily sell event tickets on your website.  Allow event owners to sell tickets while you collect a commission from sales.

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater
