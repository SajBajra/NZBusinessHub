== Description ==

Allows users to save search and users will receive an email notification when new listings published matching saved search.

== Requirements ==
* GeoDirectory 2.3 or higher - https://wordpress.org/plugins/geodirectory/
* GeoDirectory Advanced Search Filters 2.3 or higher - https://wpgeodirectory.com/downloads/advanced-search-add-on/
* BootStrap Design Style active

== Installation ==

1. Upload 'geodir-save-search-notifications' directory to the '/wp-content/plugins/' directory
2. Activate the plugin "GeoDirectory Save Search Notifications" through the 'Plugins' menu in WordPress
3. Go to WordPress Admin -> GeoDirectory -> Settings > Emails -> Save Search Notifications & GeoDirectory -> Settings > Advance Search > Saved Search Notifications to customize behaviour as needed.
