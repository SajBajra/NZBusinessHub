Allows users to build an embedded ratings widget for their listing to show on their own website.

== Description ==

Allows users to build an embedded widget to show on their own website.
Let users show they listing and current rating on their own site.
Let users build their widget to match the style of their own site.

= Inspired by Trip Advisor rating widget =

The Embeddable Ratings Badge allows listing owners to create a widget to display their live ratings from your directory on their website. Similar to the Trip Advisor rating widget.

Your directory badge on the userâ€™s site is a social proof that will give visitors viewing it more confidence in the business and the ability to view specific rating details.

= Key Features =

* Limit who can build embed code; everyone, logged in users or owner.
* All design settings can be locked by the admin or given to the user to set.
* Widget can be styled by the user to match the style of their site.

== Installation ==

1. Upload 'geodir-embed' directory to the '/wp-content/plugins/' directory
2. Activate the plugin "GeoDirectory Embed" through the 'Plugins' menu in WordPress

= Minimum Requirements =

* WordPress 5.0 or later
* GeoDirectory 2.3 or later - https://wordpress.org/plugins/geodirectory/
