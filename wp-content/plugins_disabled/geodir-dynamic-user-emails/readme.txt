Allows to send an instant emails to the users. @todo

== Requirements ==
* GeoDirectory 2.3 or higher - https://wordpress.org/plugins/geodirectory/
* BootStrap Design Style mode active

== Installation ==

1. Upload 'geodir-dynamic-user-emails' directory to the '/wp-content/plugins/' directory
2. Activate the plugin "GeoDirectory Dynamic User Emails" through the 'Plugins' menu in WordPress
3. Go to WordPress Admin -> GeoDirectory -> Settings -> Dynamic Emails to customize behaviour as needed.
